# 杠杆轮动投资策略 — 可落地执行方案

> 基于 2016-2026 十年回测数据 | 生成于 2026-06-04 | 版本 v1.0

---

## 一、核心结论（TL;DR）

**"3倍杠杆 + 轮动 + 止损"的核心价值是避坑保底，而非追求最高收益。**

> ⚠️ 以下所有「打败B&H」均指**同杠杆 3x B&H 中表现更差的那只**——公平对比，只比轮动机制是否增值。

| 结论 | 数据支撑 |
|------|----------|
| ❌ 死拿单只股票杠杆 → 不是最优 | 4只股票10年最优杠杆仅 2.0x–3.0x，无人扛住 10x |
| ❌ 5x/6x 高杠杆 → 敲出风险过高 | 5x KO 率 20%–40%，6x KO 率 40%–60% |
| ❌ 不加轮动的杠杆 → 必然敲出 | 每只股票都有 -40% 以上的年份 |
| ✅ **3x + 20% 止损轮动** → 保底最优 | 80%–90% 年份打败**同杠杆下表现差的那只**（避坑） |
| ⚠️ 轮动不等于超额收益 | 轮动很少打败**同杠杆下表现好的那只**（10%–44%） |

## 二、理论基石

### 2.1 波动率衰减公式

杠杆产品的每日重置导致收益偏离简单倍数：

$$\ln\frac{V_T}{V_0} = L \cdot \sum r_i - \frac{L^2}{2} \cdot \sum r_i^2$$

- 第一项是期望收益（线性放大）
- **第二项是衰减项**（随 L² 增长，不可消除）
- 当 L 超过 Kelly 最优值 L* = μ/σ² 时，收益开始下降甚至转负

### 2.2 四只股票 Kelly 最优杠杆

| 股票 | 年化 μ | 年化 σ | Kelly L* | 实际最优 | 10x 结果 |
|------|--------|--------|----------|----------|----------|
| NVDA | 79% | 48% | 3.4x | 2.5x | KO (2022-06) |
| MU | 35% | 52% | 1.3x | 2.0x | KO (2025-04) |
| GOOGL | 28% | 28% | 3.6x | 3.0x | KO (2022-10) |
| AMZN | 31% | 32% | 3.0x | 2.5x | KO (2022-12) |

> **关键发现：理论 Kelly 值比实际最优高出 0.5–2.3x。实操中必须下调杠杆，因为 Kelly 假设连续复利，而真实世界有隔夜跳空、流动性冲击和尾部风险。**

## 三、策略设计

### 3.1 核心策略：3x 杠杆轮动（主力策略）

```
┌─────────────────────────────────────────────────────┐
│  入场：将资金分成 2 等份                               │
│  标的：NVDA(3x) + MU(3x)                             │
│  规则：当任一仓位从最高点回撤 ≥ 20%                     │
│        → 立即卖出，全仓切换至另一标的                    │
│        → 重置最高点追踪                                │
│  退出：不再主动退出，持续轮动                            │
└─────────────────────────────────────────────────────┘
```

**参数选择理由**：
- **3x 杠杆**：10 年零敲出率最高（4/6 组合 0% KO），收益/风险比最优
- **20% 阈值**：在反应速度和避免过度交易间的最优平衡。15% 过于敏感（噪音多），25%+ 反应太慢
- **NVDA-MU 为第一优先级**：十年平均 +962% 碾压所有其他组合

### 3.2 增强策略：双组合并行（分散化）

在 NVDA-MU 基础上，增加第二组轮动：

```
┌─────────────────────────────────────────────────────┐
│  组合 A（60% 资金）：NVDA ↔ MU    3x 杠杆  20% 阈值   │
│  组合 B（40% 资金）：NVDA ↔ GOOGL  3x 杠杆  20% 阈值   │
│                                                      │
│  理由：                                              │
│  - NVDA-MU 收益最高但含 MU 个股风险                    │
│  - NVDA-GOOGL 最稳定（8/10 年打败差股票，KO 0/10）    │
│  - 两组共用 NVDA 但对手方不同，实现风险分散             │
└─────────────────────────────────────────────────────┘
```

### 3.3 保守策略：单组合宽阈值（低风险偏好）

```
┌─────────────────────────────────────────────────────┐
│  组合：GOOGL ↔ AMZN   3x 杠杆  25% 阈值               │
│                                                      │
│  适用场景：                                          │
│  - 对回撤容忍度低                                     │
│  - 不希望单只股票权重过高                              │
│  - GOOGL/AMZN 相关性高，轮动频率低（~3次/年）         │
│  - 十年 0% KO，最安全但收益也最低（+525%）            │
└─────────────────────────────────────────────────────┘
```

## 四、风险管理体系

### 4.1 绝对不能做的事

| 禁止事项 | 原因 |
|----------|------|
| 🚫 使用 5x 以上杠杆 | 10 年间每只股票都有敲出记录 |
| 🚫 单吊一只股票加杠杆 | 2022 NVDA -50%，单吊 3x 直接 -90% |
| 🚫 在熊市确认后继续持有 | 2022 年所有 6x 杠杆全部 KO |
| 🚫 用全部资金做一组轮动 | 黑天鹅来临时没有容错空间 |

### 4.2 风控规则

```
┌──────────────────────────────────────────────────────┐
│  1. 仓位上限：单组合不超过总资金 60%                    │
│  2. 组合止损：如果两组轮动同时亏损超 30%，暂停交易 1 个月 │
│  3. 年度审视：每年 6 月回测一次，调整阈值（±5%）         │
│  4. 现金储备：始终保持 ≥ 20% 现金，不在极端行情满仓       │
│  5. 杠杆上限：任何情况下单标的杠杆 ≤ 3x                 │
└──────────────────────────────────────────────────────┘
```

### 4.3 极端情景压力测试

| 情景 | 历史参照 | 预期影响 | 应对 |
|------|----------|----------|------|
| NVDA 暴跌 50% | 2022 年已发生 | 3x 轮动跌 37% 后切换至对手方 | 20% 阈值触发，自动止损 |
| MU 暴跌 44% | 2018-2019 已发生 | 3x 轮动跌 31% 后切换 | 轮动生效，保护本金 |
| 两股同时暴跌 | 2022 科技股普跌 | 换仓可能无效 | 触发 30% 组合止损，暂停交易 |
| 连续敲出 | 2018+2020 MU | 5x 必死，3x 幸存 | 3x 是唯一的保护 |

## 六、各年份实战指南

基于年度滚动回测的不同市场环境：

| 市场环境 | 年份示例 | 最优策略调整 |
|----------|----------|-------------|
| 🟢 **强趋势牛市** | 2016-17, 2023-24 | 降低阈值至 15%，多赚趋势 |
| 🟡 **震荡分化** | 2020-21, 2024-25 | 标准 20% 阈值，正常轮动 |
| 🔴 **熊市/普跌** | 2018-19, 2022-23 | 提高阈值至 30%，减少无意义换仓 |
| ⚡ **极端波动** | 2025-26 (MU +957%) | 收紧至 15%，快速止盈锁定 |

> **实操建议**：每年 6 月审视过去一年市场环境，用回测脚本验证阈值是否需要调整。

## 七、执行清单（每月操作）

```
每月 1 日：
  □ 检查两组轮动的当前持仓
  □ 计算当前回撤（vs 最高点）
  □ 判断是否触发 20% 阈值
  □ 如触发 → 立即切换
  □ 如未触发 → 不做任何操作
  □ 更新持仓日志

每年 6 月：
  □ 运行 annual_rolling.py 获取最新回测
  □ 审视是否需要调整阈值（±5%）
  □ 审视是否需要调整组合权重（60/40 → 50/50）
  □ 审视是否需要加入/替换标的
  □ 记录年度策略回顾
```

## 八、预期收益与风险

| 指标 | NVDA-GOOGL 最稳 | NVDA-MU 最高收益 | GOOGL-AMZN 最弱 |
|------|:---------------:|:----------------:|:---------------:|
| 十年平均年化 (rot) | ~250% | ~962% | ~59% |
| 最好年份 | +949% (2016-17) | +3,823% (2025-26) | +390% (2025-26) |
| 最差年份 (不KO) | -60% (2018-19) | -100% (2024-25 KO) | -63% (2021-22) |
| KO 概率/年 | **0%** | ~10% | **0%** |
| **打败差 3x BH** ⭐ | **90%** | 78% | 30% |
| 打败好 3x BH | 30% | 44% | 10% |
| 打败差 1x BH（参考） | 80% | 70% | 50% |
| 平均年换手次数 | ~10 | ~20 | ~3 |
| 最大回撤（不敲出） | ~-60% | ~-85% | ~-63% |

> ⭐ **核心指标**：「打败差 3x BH」才是衡量轮动是否有效的真正标准。在同杠杆下，轮动帮你避开了最差的持仓——这就是它的全部价值。
